== Description ==

Extension to Leaflet Maps Marker to make markers from images.

An option is added to the image menu to create a marker from the image.
The location info in the image is used to define the location of the marker.

Tested with:

* WordPress: 3.7 to 4.1
* [Leaflet Maps Marker](https://wordpress.org/plugins/leaflet-maps-marker/): 3.9.5
* [Maps Marker Pro](https://www.mapsmarker.com/): 1.9.1
* [NextGEN Gallery](https://wordpress.org/plugins/nextgen-gallery/): 2.0.66.37

Please contact me with suggestions for other gallery plugins that I could support.


== Installation ==

Login to your WordPress dashboard, go to Plugins >> Add New, search for image-marker, and click to install.


== Frequently Asked Questions ==

= Why do images in NextGEN gallery not have location info? =

When you upload the image, do not resize it since that removes the location info.
